Binary Search Tree

Welcome to my implementation of a BST! I made this project to help me learn the basics of JavaScript and HTML. Navigating through this web page allows you to perform all of the basic functions on a binary search tree, which I have implemented in JavaScript.

All code was written by Shawn O'Grady

Some important notes about this program:

  1. Each value in the tree needs to be a number
    • Trying to add a non-number value like hello or one will trigger a pop-up
  2. Each value in the tree must be unique
    • i.e. no duplicate values allowed
    • trying to add a duplicate will trigger a pop-up
  3. The text under the Last action: field displays the most recent valid operation on the tree
    • I am defining an invalid operation as the following:
      1. entering a non-numeric input to any of the fields
      2. trying to insert a duplicate value to the tree
      3. trying to search for a value or remove a value from an empty tree
      4. trying to print an empty tree
      5. trying to print the tree with out selecting a printing order
    • Any of the above will trigger a pop-up
